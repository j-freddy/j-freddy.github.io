# My Personal Site

Made with [Jekyll][2]. Hosted on [GitHub Pages][1].

## Install

### Prerequisites

This site is made with Jekyll. Refer to prerequisites section of Jekyll
documentation [here][3].

### Run locally

```sh
# Install Gems
bundle install
# Windows
jekyll serve
# Mac
bundle exec jekyll serve
```
